全排列（无重复元素）

    比较简单，直接递归即可；

全排列（有重复元素）

    方法在于，递归构建新的vector时，不能在同一位置填入和上次相同的数字。设置方法为，在递归的循环体中，设置检测：如果本位的数和下一位的相同，则跳过本位。

子集（无重复元素）
    还是采用递归的方法，关键在于要设置一个全局的hash_table（unordered_set）,用于记录哪些组合已经被使用过。

子集（有重复元素）
    在无重复元素的子集的基础上，设置有重复元素的检测方法（全排列中使用的方法）
